# HTML/CSS/JS Quiz

**Challenge your mind with fun and engaging questions!**  

This is a **web-based quiz project** built using **HTML, CSS, and JavaScript**. The quiz contains 20 interactive questions about **HTML, CSS, and JavaScript**. Users can select answers and test their knowledge of web development.

---

## Features

- 20 multiple-choice questions  
- Instant feedback (correct/incorrect)  
- Responsive and simple design  
- Built entirely with HTML, CSS, and JavaScript
